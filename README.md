# Video Capture — AI 智能混剪工具

Video Capture 是一款桌面端视频处理工具，集成了视频下载、语音识别、AI 文案生成、自动混剪等功能，帮助内容创作者高效完成视频二次创作。

## 三、基本使用

### 1. 下载视频

在「原始视频管理」页面，点击「下载」按钮，输入视频链接即可下载。支持 YouTube、B站等主流平台。

### 2. 语音转文字

上传或下载视频后，系统会自动提取音频并进行语音识别，生成带时间戳的字幕文本。

### 3. AI 文案处理

选中视频，点击「编辑」，可以使用 AI 对字幕进行：
- **内容扩展**：丰富原文内容
- **字幕优化**：合并断句、去除冗余
- **语义分析**：提取关键信息

### 4. 素材管理

将处理后的视频片段保存为素材，支持：
- 手动裁剪片段
- TTS 文字转语音生成音频素材
- AI 智能拆分

### 5. 自动混剪

在「混剪视频管理」页面：
1. 点击「自动生成」进入混剪配置
2. 选择素材来源和排列方式
3. AI 自动匹配素材顺序并生成混剪视频
4. 预览满意后可导出最终视频

## 四、常见问题

**Q：启动后弹窗提示下载失败？**  
A：点击「手动安装」按说明操作，或检查网络连接后点击「重新下载」。

**Q：下载B站视频失败或合成视频失败？**  
A：由于B站下载使用you-get.exe，合成视频使用ffmpeg.exe工具，环境变量PATH中设置安装目录backend/bin目录

**Q：软件占用空间较大？**  
A：主要空间消耗在视频文件和模型上。可定期清理不需要的视频，或在「系统配置」中调整存储路径。
